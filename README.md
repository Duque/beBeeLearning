# Easy-Kit-SASS-Nunjuks
Easy and quick Kit for SASS &amp; Nunjuks

*Install:*

git clone https://github.com/Duque/Easy-Kit-SASS-Nunjuks.git

cd Easy-Kit-SASS-Nunjucks

npm install

sudo npm run watch
